# DevMind

DevMind is a full-stack AI chat application built with React, ASP.NET Core Web API, MySQL, and Ollama.

The project was created as a learning-focused but practical app to improve both frontend and backend development skills, while also integrating a local AI model without using paid APIs.

## Features

- Create and manage chat conversations
- Send and display messages in real time
- Store conversations and messages in MySQL
- Generate AI responses using a local Ollama model
- Connect React frontend to ASP.NET Core backend
- Persist both user and assistant messages in the database

## Tech Stack

### Frontend
- React
- TypeScript
- Vite
- CSS

### Backend
- ASP.NET Core Web API
- Entity Framework Core
- MySQL
- Ollama (local AI model)
